## Hypothesis

An adversary with user-level code execution is enumerating installed browser extensions by reading extension manifest files or abusing native messaging IPC — identifying high-value extensions (password managers, SSO plugins) for targeted exploitation.

---

## Hunt Queries

### Hunt 1 — Non-Browser Process Reading Multiple Extension Manifests

```sql
SELECT process_name, process_pid, COUNT(DISTINCT file_path) as manifests_read,
       array_agg(DISTINCT file_path) as manifest_paths
FROM file_access_events
WHERE file_path ILIKE '%Extensions%manifest.json%'
AND process_name NOT IN ('chrome.exe', 'firefox.exe', 'msedge.exe', 'brave.exe',
                          'Code.exe', 'electron.exe')
AND timestamp >= NOW() - INTERVAL 30 DAYS
GROUP BY process_name, process_pid
HAVING manifests_read > 3
ORDER BY manifests_read DESC
```

### Hunt 2 — Native Messaging Host Spawned by Non-Browser Process

```sql
SELECT parent_process_name, child_process_name, child_process_path,
       command_line, timestamp
FROM process_creation_events
WHERE child_process_path ILIKE '%NativeMessagingHosts%'
AND parent_process_name NOT IN ('chrome.exe', 'firefox.exe', 'msedge.exe')
AND timestamp >= NOW() - INTERVAL 30 DAYS
ORDER BY timestamp DESC
```

### Hunt 3 — Browser Profile Directory Mass Access

```sql
SELECT process_name, COUNT(DISTINCT file_path) as files_accessed
FROM file_access_events
WHERE file_path ILIKE '%Google\Chrome\User Data%'
   OR file_path ILIKE '%Mozilla\Firefox\Profiles%'
   OR file_path ILIKE '%Microsoft\Edge\User Data%'
AND process_name NOT IN ('chrome.exe', 'firefox.exe', 'msedge.exe',
                          'MicrosoftEdge.exe', 'SearchIndexer.exe')
AND timestamp >= NOW() - INTERVAL 30 DAYS
GROUP BY process_name
HAVING files_accessed > 10
ORDER BY files_accessed DESC
```
