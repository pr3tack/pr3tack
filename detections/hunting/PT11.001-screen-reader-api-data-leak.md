## Hypothesis

An adversary with Accessibility API access is systematically traversing the UI element trees of multiple applications — harvesting rendered screen content (web app data, internal tool output, document text) for exfiltration, without capturing screenshots or accessing the filesystem.

---

## Hunt Queries

### Hunt 1 — Process Accessing Accessibility Tree of 3+ Distinct Applications

```sql
SELECT process_name, process_pid,
       COUNT(DISTINCT target_process_name) as app_count,
       array_agg(DISTINCT target_process_name) as target_apps
FROM accessibility_api_events
WHERE api_call ILIKE '%GetRootElement%'
   OR api_call ILIKE '%FindAll%'
   OR api_call ILIKE '%AXChildren%'
AND process_name NOT IN ('nvda.exe','JAWS','narrator.exe','ZoomText','magnify.exe')
AND timestamp >= NOW() - INTERVAL 30 DAYS
GROUP BY process_name, process_pid
HAVING app_count >= 3
ORDER BY app_count DESC
```

### Hunt 2 — High-Frequency Background Accessibility Polling

```sql
SELECT process_name, process_pid,
       COUNT(*) as api_calls_per_hour,
       has_visible_window
FROM accessibility_api_events
WHERE has_visible_window = false  -- Background process
AND timestamp >= NOW() - INTERVAL 30 DAYS
GROUP BY process_name, process_pid, DATE_TRUNC('hour', timestamp)
HAVING api_calls_per_hour > 500
ORDER BY api_calls_per_hour DESC
```

### Hunt 3 — Accessibility Read Followed by Outbound Network Connection

```sql
SELECT a.process_name, a.process_pid,
       COUNT(DISTINCT a.target_process_name) as apps_accessed,
       n.dst_ip, n.dst_port, n.timestamp as network_timestamp
FROM accessibility_api_events a
JOIN network_connections n ON a.process_pid = n.src_pid
WHERE a.timestamp >= n.timestamp - INTERVAL 5 MINUTES
AND a.timestamp <= n.timestamp
AND a.process_name NOT IN ('nvda.exe','narrator.exe','JAWS')
AND n.dst_ip NOT IN (SELECT ip FROM internal_cidr_ranges)
AND a.timestamp >= NOW() - INTERVAL 30 DAYS
GROUP BY a.process_name, a.process_pid, n.dst_ip, n.dst_port, n.timestamp
HAVING apps_accessed >= 2
ORDER BY n.timestamp DESC
```
