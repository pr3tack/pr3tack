# Hunt: AI-Generated Persona Synthesis — Synthetic Identity Signal Correlation

## Metadata

| Field | Value |
|---|---|

| **Tactic** | PT02 — Resource Development |
| **Detection Type** | Hunt |
| **Data Source** | Identity Verification Logs / HR Systems / Recruitment Platform Logs / Email Gateway / Communication Platforms |
| **Confidence** | Low–Medium |
| **Status** | Draft |

---

## Hypothesis

An adversary has synthesised one or more AI-generated personas — complete with fabricated professional histories, AI-generated profile images, and plausible writing styles — and is using them to establish trusted relationships within the organisation (candidates, vendors, contractors, researchers). Evidence will appear as a cluster of synthetic identity signals across onboarding, communication, and trust-establishment workflows.

---

## Hunt Scope

| Scope Item | Detail |
|---|---|
| **Time window** | Last 90 days |
| **Data sources** | Recruitment platform logs (LinkedIn, Greenhouse, Lever), email gateway metadata, video conferencing logs, identity verification outputs, contractor/vendor onboarding records, background check results |
| **Key questions** | Are any recently onboarded individuals or candidates flagged by AI-content detection tooling? Do any profile images match known GAN/diffusion model artefact signatures? Are there identity inconsistencies between stated history and verifiable records? Do any communication patterns show AI-writing style signatures? |
| **Priority targets** | External candidates, new contractors, new vendor contacts, inbound research partnership requests |

## Hunt Queries

### Hunt 1 — AI-Content Detection Flag Correlation

Correlate AI-content detection tool output across inbound communications and profile submissions:

```sql
SELECT identity_id, identity_name, submission_type,
       ai_content_score, ai_image_score, submission_date,
       reviewer_notes
FROM identity_submissions
WHERE (ai_content_score > 0.75 OR ai_image_score > 0.80)
AND submission_date >= NOW() - INTERVAL 90 DAYS
ORDER BY ai_content_score DESC, ai_image_score DESC
```

### Hunt 2 — Profile Image Artefact Analysis

Hunt for profile images submitted through onboarding or recruitment workflows that match GAN/diffusion model artefact signatures (requires integration with an image analysis service):

```sql
SELECT identity_id, image_url, gan_probability_score,
       diffusion_artefact_score, image_submitted_date
FROM profile_image_analysis_log
WHERE (gan_probability_score > 0.70 OR diffusion_artefact_score > 0.70)
AND image_submitted_date >= NOW() - INTERVAL 90 DAYS
ORDER BY GREATEST(gan_probability_score, diffusion_artefact_score) DESC
```

### Hunt 3 — Identity Consistency Cross-Check

Hunt for onboarded individuals or candidates where stated employment history or education cannot be verified through standard background check channels:

```sql
SELECT identity_id, stated_employer, stated_role, stated_dates,
       verification_status, verification_notes, onboarding_date
FROM background_check_results
WHERE verification_status IN ('unverifiable', 'inconsistent', 'employer_not_found')
AND onboarding_date >= NOW() - INTERVAL 90 DAYS
ORDER BY onboarding_date DESC
```

### Hunt 4 — Writing Style Consistency Analysis

Hunt for individuals whose written communications show consistent AI-writing style signatures across multiple messages — unusually uniform sentence structure, rare use of contractions, or statistically unlikely vocabulary consistency:

```sql
SELECT sender_identity, COUNT(*) as message_count,
       AVG(ai_writing_score) as mean_ai_score,
       STDDEV(ai_writing_score) as stddev_ai_score,
       MIN(ai_writing_score) as min_score
FROM email_gateway_analysis
WHERE message_date >= NOW() - INTERVAL 90 DAYS
AND sender_type = 'external'
GROUP BY sender_identity
HAVING mean_ai_score > 0.70
AND message_count >= 5
ORDER BY mean_ai_score DESC
```

### Hunt 5 — Multi-Signal Persona Correlation

Identify individuals who trigger multiple synthetic identity signals simultaneously — the strongest indicator of a fabricated persona:

```sql
SELECT i.identity_id, i.identity_name, i.submission_date,
       CASE WHEN a.ai_content_score > 0.75 THEN 1 ELSE 0 END as ai_content_flag,
       CASE WHEN p.gan_probability_score > 0.70 THEN 1 ELSE 0 END as ai_image_flag,
       CASE WHEN b.verification_status = 'unverifiable' THEN 1 ELSE 0 END as unverifiable_history,
       CASE WHEN e.mean_ai_score > 0.70 THEN 1 ELSE 0 END as ai_writing_flag
FROM identity_submissions i
LEFT JOIN identity_submissions a ON i.identity_id = a.identity_id
LEFT JOIN profile_image_analysis_log p ON i.identity_id = p.identity_id
LEFT JOIN background_check_results b ON i.identity_id = b.identity_id
LEFT JOIN (SELECT sender_identity, AVG(ai_writing_score) as mean_ai_score
           FROM email_gateway_analysis GROUP BY sender_identity) e
  ON i.identity_name = e.sender_identity
WHERE i.submission_date >= NOW() - INTERVAL 90 DAYS
HAVING (ai_content_flag + ai_image_flag + unverifiable_history + ai_writing_flag) >= 2
ORDER BY (ai_content_flag + ai_image_flag + unverifiable_history + ai_writing_flag) DESC
```

## Expected Findings

| Finding Type | Likely Explanation | Priority |
|---|---|---|
| Profile image GAN score > 0.85 | AI-generated headshot — very high confidence synthetic identity | High |
| AI content score > 0.80 across multiple messages | AI-written communications — may be legitimate AI-assisted writing | Medium |
| Unverifiable employment history + AI image flag | Multi-signal synthetic persona indicator | Critical |
| 3+ signals triggered for same identity | High-confidence fabricated persona — escalate immediately | Critical |

---

## Tuning Notes

- **Tool dependency:** This hunt requires AI-content detection tooling integrated with email, recruitment, and image processing workflows — if not deployed, prioritise tooling procurement before operationalising this hunt
- **False positives:** AI-assisted writing tools (Grammarly, Copilot, ChatGPT) are widely used legitimately — Hunt 4 findings alone are insufficient for escalation without corroborating signals
- **Privacy considerations:** Ensure AI analysis of communications complies with your jurisdiction's data protection requirements and employee/candidate privacy policies before deployment
- **Legitimate use of synthetic content:** Some legitimate roles (privacy researchers, security professionals) may use pseudonymous identities — validate findings against HR context before escalating

## Validation

1. Submit a test candidate profile using a known AI-generated image (e.g., from thispersondoesnotexist.com) through your recruitment platform
2. Confirm Hunt 2 identifies the GAN-generated image with high probability score
3. Submit AI-generated cover letter text and confirm Hunt 1 flags the AI content score
4. Verify Hunt 5 correlates both signals for the same test identity
