# Tactics

PR3TACK aligns techniques to tactical objectives similar to established models.

## Current Tactics

| ID | Tactic | Description |
|---|---|---|
| PT01 | Reconnaissance | Pre-connection and identity-centric footprinting of the target's SaaS, cloud, and AI attack surface |
| PT02 | Resource Development | Creation and incubation of trusted infrastructure, synthetic personas, and adversary capabilities |
| PT03 | Execution | Novel execution paths leveraging underexplored OS, firmware, and peripheral primitives |
| PT04 | Persistence | Unconventional persistence mechanisms using obscure OS subsystems not routinely audited |
| PT05 | Privilege Escalation | Emerging escalation paths via driver abuse, credential store manipulation, and trust chain exploitation |
| PT06 | Supply Chain Preparation | Manipulation of the developer environment and dependency ecosystem prior to active compromise |
| PT07 | Defense Evasion | Techniques that exploit gaps in cloud metadata pipelines, telemetry agents, and detection stacks |
| PT08 | Command & Control | Covert C2 channels abusing legitimate services, protocols, and physical-layer interfaces |
| PT09 | Discovery | Internal enumeration techniques targeting identity topology, IPC channels, and browser state |
| PT10 | Collection | Novel data collection paths via OS clipboard history, accessibility APIs, and credential stores |
| PT11 | Exfiltration | Covert exfiltration via accessibility APIs and victim-owned cloud storage staging |
| PT12 | Unix Contextual Stealth | Unix-specific techniques for stealthy staging and credential harvesting using OS-native primitives |
| PT13 | Lateral Movement | Propagation via compromised software update channels and permissive cross-unit policy inheritance |
| PT14 | Pre-Positioning | Long-horizon staging via OSS sleeper commits, domain aging, and infrastructure pre-registration |
| PT15 | Resilience Erosion | Degradation of recovery capability via backup corruption and patch-fatigue manipulation |
| PT16 | Governance Subversion | Adversarial influence over standards bodies, procurement processes, and trust frameworks |
| PT17 | Cognitive Manipulation | Exploitation of analyst psychology via habituation lures and adversarial log crafting |
| PT18 | Operational Tempo Manipulation | Timing attacks that exploit staffing gaps, holiday windows, and automated cost responses |
| PT19 | AI/ML Subversion | Attacks against machine learning pipelines via training data poisoning and adversarial inputs |
| PT20 | Sociotechnical Pressure | Coordinated external pressure campaigns that coerce security-adverse organisational decisions |
| PT21 | Digital Exhaust Manipulation | Pollution and flooding of threat intelligence feeds and telemetry pipelines |

---

Each tactic contains techniques that describe **how an attacker could achieve that objective in a novel or underexplored way**.
